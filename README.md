# Wheels Enchntment - Premium Skating Products

A modern, minimal, and scalable static frontend website built with React, Vite, and Material UI.

## 🎨 Brand Identity

- **Primary Color**: #0B1F3A (Deep Navy Blue)
- **Secondary Color**: #E91E63 (Vibrant Pink)
- **Background**: #FFFFFF / #F8F9FB
- **Design Philosophy**: Premium, sporty, minimal aesthetic inspired by Nike/Adidas

## 🚀 Tech Stack

- React 18
- Vite
- Material UI (MUI)
- React Router
- Emotion (CSS-in-JS)

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📄 Pages

- **Home**: Hero section, featured products, features, CTA
- **Products**: Product grid with filters and sorting
- **Gallery**: Image gallery with category filters
- **About**: Brand story, mission, vision, values
- **Contact**: Contact form and information

## 🧩 Components

- Navbar (responsive with mobile drawer)
- Footer (dark navy with links and social icons)
- ProductCard (clean card with hover effects)
- SectionHeader (consistent section titles)

## 🎯 Features

- Fully responsive design
- Clean, minimal UI
- Smooth hover transitions
- Mobile-first approach
- Static data (no backend required)
- Professional brand aesthetic

## 📁 Project Structure

```
src/
├── components/     # Reusable components
├── pages/          # Page components
├── theme/          # MUI theme configuration
├── data/           # Static data
├── assets/         # Images and static files
├── App.jsx         # Main app component
└── main.jsx        # Entry point
```

## 🎨 Design Guidelines

- Use pink (#E91E63) only as accent color
- Maintain clean white + navy base
- Strong whitespace and grid-based layouts
- Subtle shadows, no heavy effects
- Medium border radius (8-12px)
- Professional, premium feel
